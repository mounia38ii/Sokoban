## Sokoban – Jeu & Intelligence Artificielle (Java)
Ce projet implémente le jeu Sokoban en Java avec :
Une interface console simple pour jouer.
Une IA capable de résoudre automatiquement les niveaux.
Architecture modulaire et lisible pour les composants du jeu et de l’IA.

## Fonctionnalités principales
Déplacement des personnages et gestion des boîtes (mécanique Sokoban classique).
IA intégrée pour résoudre les niveaux automatiquement.
Gestion de niveaux multiples.
Mode console simple et intuitif.
## Compilation & exécution
1. Compiler le projet
javac -d out -cp src src/**/*.java
Cette commande compile tous les fichiers Java dans le dossier src et place les .class dans le dossier out.
2. Lancer le jeu
java -cp out Sokoban
